# AI Resume Builder

AI Resume Builder is a web application that helps students generate professional resumes automatically using Artificial Intelligence.

## Features
- User Login and Registration
- AI Resume Generation
- Multiple Resume Templates
- User Dashboard
- MySQL Database Integration

## Tech Stack

Frontend
- React.js
- Vite
- CSS

Backend
- Python
- Streamlit

Database
- MySQL

AI Model
- Transformer based AI model
